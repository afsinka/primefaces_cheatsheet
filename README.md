# primefaces_cheatsheet

- Application Form - 1 with PrimeFaces validations and captcha
	* First Name (text field)
	* Last Name (text field)
	* Date of Birth (calendar)
	* Place of Birth (autocomplete)
	* GPA (float value)
	* City (Drop-down list)

	* First Name (text field)
	* Last Name (text field)
	* Date of Birth (calendar)
	* Place of Birth (autocomplete)
	* GPA (float value)
	* City (Drop-down list)
